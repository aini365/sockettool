# 介绍
SocketTool是一款使用javafx编写的跨平台的socket工具。作者一直在windows环境下使用sockettool，苦于mac环境下不能使用，原型参考windows版本的sockettool,编写了此程序。

# 功能

+ tcp服务端和客户端
+ udp服务端和客户端
+ 支持ezsp串口协议解析
